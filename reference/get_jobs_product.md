# Job-related API functions Returns data on all jobs for a given product

Corresponds to, for example: https://api.municode.com/Jobs/product/12429
Returns all historical versions of a product (e.g., all historical
versions of the zoning ordinance)

## Usage

``` r
get_jobs_product(product_id)
```

## Arguments

- product_id:

  A unique identifier for a product
