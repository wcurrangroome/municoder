# Get information about a given node's ancestor(s)

Corresponds to, for example:
https://api.municode.com/codesToc/breadcrumb?jobId=426172&nodeId=THZOORALVI&productId=12429

## Usage

``` r
get_codes_toc_breadcrumb(job_id, node_id, product_id)
```

## Arguments

- job_id:

  A unique identifier for a job

- node_id:

  A unique identifier for a node within the specified product
  (ordinance)

- product_id:

  A unique identifier for a product
